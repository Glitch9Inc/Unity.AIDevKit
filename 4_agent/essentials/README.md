# Essentials

Agent의 핵심 기능들을 다루는 필수 가이드입니다. 이 섹션에서는 Agent 설정부터 대화 관리, 응답 생성, 이벤트 처리까지 Agent를 효과적으로 활용하는데 필요한 모든 기본 개념을 다룹니다.

## 주요 내용

### Agent Configuration

Agent의 기본 설정, API 타입, 대화 저장소, 오디오 설정, 메모리 관리 등 Agent를 구성하는 방법을 설명합니다.

### Agent Lifecycle

Agent의 초기화, 상태 관리, 종료까지 생명주기 전반을 다룹니다.

### Conversation Management

대화 생성, 로드, 저장 및 메시지/아이템 관리 방법을 설명합니다.

### Generating Responses

텍스트 메시지 생성, 스트리밍, 파일 첨부, 오디오 입력, 요청 취소 등 다양한 응답 생성 방법을 다룹니다.

### Parameters

모델, 음성, 도구, 온도, 최대 토큰 등 Agent의 파라미터 설정 방법을 설명합니다.

### Audio

입력/출력 오디오 처리, 음성 인식, 텍스트 음성 변환 등 오디오 기능을 다룹니다.

### Events & Hooks

Agent의 다양한 이벤트와 훅을 활용하여 반응형 시스템을 구축하는 방법을 설명합니다.

### Advanced Topics

Agent 서비스, 컨트롤러 아키텍처, 이벤트 라우터, 커스텀 채팅 서비스 등 고급 주제를 다룹니다.
