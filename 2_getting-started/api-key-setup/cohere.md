# Cohere

## Getting Your API Key

1. Visit [Cohere Dashboard](https://dashboard.cohere.com)
2. Sign up or log in
3. Go to **API Keys** section
4. Copy your API key or create a new one

{% hint style="info" %}
Cohere offers a free tier for testing.
{% endhint %}

## Adding to AI Dev Kit

**Tools > AI Dev Kit > Settings**

Find **Cohere** section and paste your API key.

## Pricing

Check [Cohere Pricing](https://cohere.com/pricing) for current rates.
