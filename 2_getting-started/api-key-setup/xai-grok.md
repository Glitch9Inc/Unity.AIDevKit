# xAI (Grok)

## Getting Your API Key

1. Visit [xAI Console](https://console.x.ai)
2. Sign up or log in
3. Go to **API Keys** section
4. Click **Create new key**
5. Copy your API key (starts with `xai-`)

{% hint style="info" %}
Grok is xAI's conversational AI with real-time knowledge.
{% endhint %}

## Adding to AI Dev Kit

**Tools > AI Dev Kit > Settings**

Find **xAI (Grok)** section and paste your API key.

## Available Models

- `XAIModel.Grok` - Latest Grok model
- `XAIModel.GrokVision` - Vision-capable Grok
