# Perplexity

## Getting Your API Key

1. Visit [Perplexity API](https://www.perplexity.ai/settings/api)
2. Sign up or log in
3. Go to **API** section
4. Click **Generate API Key**
5. Copy your API key (starts with `pplx-`)

{% hint style="info" %}
Perplexity provides AI with real-time web search capabilities.
{% endhint %}

## Adding to AI Dev Kit

**Tools > AI Dev Kit > Settings**

Find **Perplexity** section and paste your API key.

## Available Models

- `PerplexityModel.Sonar` - Online search-enabled
- `PerplexityModel.SonarHuge` - Larger search model
- `PerplexityModel.Codellama70B` - Code generation
