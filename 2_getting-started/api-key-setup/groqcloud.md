# GroqCloud

## Getting Your API Key

1. Visit [GroqCloud Console](https://console.groq.com)
2. Sign up or log in
3. Go to **API Keys** section
4. Click **Create API Key**
5. Copy your API key (starts with `gsk_`)

{% hint style="info" %}
GroqCloud offers extremely fast inference speeds.
{% endhint %}

## Adding to AI Dev Kit

**Tools > AI Dev Kit > Settings**

Find **GroqCloud** section and paste your API key.

## Available Models

- `GroqModel.Llama3_70B` - Llama 3 70B
- `GroqModel.Llama3_8B` - Llama 3 8B
- `GroqModel.Mixtral8x7B` - Mixtral 8x7B
- `GroqModel.Gemma7B` - Gemma 7B

## Pricing

Check [GroqCloud Pricing](https://console.groq.com/pricing) for current rates.
