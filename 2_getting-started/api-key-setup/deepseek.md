# DeepSeek

## Getting Your API Key

1. Visit [DeepSeek Platform](https://platform.deepseek.com)
2. Sign up or log in
3. Go to **API Keys** section
4. Click **Create new secret key**
5. Copy your API key (starts with `sk-`)

{% hint style="warning" %}
Store your API key securely. You won't be able to see it again.
{% endhint %}

## Adding to AI Dev Kit

**Tools > AI Dev Kit > Settings**

Find **DeepSeek** section and paste your API key.

## Pricing

Check [DeepSeek Pricing](https://platform.deepseek.com/pricing) for current rates.
