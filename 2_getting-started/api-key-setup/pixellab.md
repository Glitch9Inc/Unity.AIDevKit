# PixelLab

## Getting Your API Key

1. Visit [PixelLab Dashboard](https://pixellab.ai/dashboard)
2. Sign up or log in
3. Go to **API Keys** section
4. Click **Generate API Key**
5. Copy your API key

{% hint style="info" %}
PixelLab specializes in AI-powered image generation for games.
{% endhint %}

## Adding to AI Dev Kit

**Tools > AI Dev Kit > Settings**

Find **PixelLab** section and paste your API key.

## Available Models

- Specialized game asset generation
- Character sprite generation
- Texture generation
- Pixel art generation
